## Installation

1. Clone Github Repository

   ```sh
   git clone https:://github.com/sovware/helpgent.git
   ```

2. Install composer dependencies
   ```sh
   composer install --no-cache
   ```

3. Install npm dependencies (Node v18.x)
   ```sh
   npm install
   ```

4. Compile Scripts
   ```sh
   npm start --no-watch
   ```

4. Build plugin
   ```sh
   npm run build
   ```
